FAQ

Common questions.

Which banks are supported?
Any of the 2,500+ banks across 29 European countries supported by Enable Banking, including Revolut, N26, Monzo, Wise, Millennium BCP, Santander, ING, BNP Paribas, and many more. Personal accounts only.
Do I need to know how to code?
No coding needed. Setup takes about 10 minutes — we walk you through every command step by step. The rest is done through a browser-based setup wizard.
Is my bank data safe?
Yes. Klartion uses read-only access via Enable Banking's regulated open banking API — it cannot move money or make any changes to your account. Your data goes directly to your machine and never passes through our servers.
What do I need to run Klartion?
A computer or server that can run Docker — a Mac, Linux machine, NAS, or a small VPS all work. Klartion is lightweight and uses very little memory or CPU.
Can I use it on more than one machine?
Each licence key can be activated on up to 2 machines. If you move to a new machine, deactivate your current installation from within the app and reactivate on the new one.
Is there a hosted version?
Not yet, but it's coming. Join the waitlist above to be notified when it launches.
